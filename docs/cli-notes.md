# CLI notes

Initial scaffold commands:
- `pire doctor`
- `pire ingest <pcap>`
- `pire summary <pcap> [--protocol <filter>]`
- `pire around <pcap> --frame <n> [--before N --after N]`
- `pire export-frames <pcap> --start <n> --end <n>`

These are intentionally minimal and are meant to validate the end-to-end shape before deeper features are added.
