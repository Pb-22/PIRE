# PIRE Markdown Templates

These are draft markdown templates for the three main PIRE knowledge categories:

- `protocol-core-template.md`
- `experiential-note-template.md`
- `current-case-template.md`

Use them as the initial content shape for:
- `library/protocols/<protocol>/core.md`
- `library/experience/<protocol-or-theme>/<note>.md`
- `library/cases/<case-id>/current-case.md`
